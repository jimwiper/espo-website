BRAND CONTEXT: espo is a fictional Japanese VTuber esports organization. 
Visual DNA: dark navy (#070914) base, neon cyan (#36f0ff) primary accent, hot magenta (#ff3ea5) secondary, amber (#f8d34a) hazard.
Aesthetic: tactical / esports / anti-idol / dark streetwear / cyberpunk-adjacent. NOT cute, NOT pastel, NOT holographic rainbow.
Typography feel: Zen Dots display font, Anton compressed headlines, JetBrains Mono for data/stats.
Motifs: crosshairs, HUD overlays, hex grids, radar pings, military patches, scanlines, glitch effects, constellation patterns.
Quality bar: premium product photography or high-end game key visual illustration quality. No cheap mockups.
Key differentiator from competitors: WHERE others use pastel aurora holographics, espo uses matte dark + selective neon glow.
